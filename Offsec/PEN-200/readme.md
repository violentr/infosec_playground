# PEN-200

**PEN-200: Penetration Testing with Kali Linux** (formerly known as PWK) is Offensive Security's foundational ethical hacking and penetration testing course. It combines written course material with hands-on practice in a virtual lab environment.

## Overview

The course teaches the core skills and mindset needed for professional penetration testing. Students learn to identify vulnerabilities, exploit systems, escalate privileges, and document their findings—simulating real-world offensive security engagements from start to finish.

## Key Topics

- Information gathering and enumeration
- Vulnerability scanning
- Web application attacks (including SQL injection)
- Client-side attacks and password attacks
- Exploit research, modification, and antivirus evasion
- Windows and Linux privilege escalation
- Pivoting, port redirection, and tunneling
- Active Directory enumeration and lateral movement
- Metasploit and basic cloud attack techniques

## Certification

Completing the course and passing the practical exam earns the **OffSec Certified Professional (OSCP)** certification—one of the most respected credentials in offensive security.

## Prerequisites

While there are no formal prerequisites, students should have:

- A solid understanding of TCP/IP networking
- Basic scripting skills (Bash and/or Python)
- Familiarity with Linux and Windows administration

## Who Is It For?

PEN-200 is designed for security professionals, network administrators, and aspiring penetration testers who want practical, hands-on training in ethical hacking and offensive security.
